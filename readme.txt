A WordPress plugin which displays recent posting frequency in a chart generated by google chart API 

== Description ==

This plugin provides a shortcode to be inserted into posts which display a nice graph showing your recent posting frequency.

= Usage =

Insert shortcode in post: [archive_chart]

= Example = 

> [archive_chart name="Posting frequency over the last 24 months" width="1200" height="400" count="24"]
